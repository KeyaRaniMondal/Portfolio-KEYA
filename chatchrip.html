ChatChrip

Live Demo GitHub

Project Overview

ChatChrip is an interactive community forum and social platform designed for global connection and knowledge exchange. It allows users to create, edit, and delete posts, participate in categorized discussions, rate content through a voting system, and engage via comments . The platform includes a secure payment system for premium features and specialized admin controls for user and content management

Key Features

  • User Authentication (JWT)
  • Category Management
  • Search Functionality
  • Post Details
  • Notifications
  • Admin Controls
  • Responsive Design

Technology Stack

React Node.js Express MongoDB Stripe JWT Axios Tailwind CSS Gemini API

System Architecture

The frontend communicates with a REST API built using Express.js. User authentication is secured with JWT, while MongoDB stores user accounts, Posts,Comments ,Notifications and Admin data.

Challenges

  • Managing authentication securely.
  • Integrating Stripe for payment processing.
  • Admin panel development.
  • Maintaining responsive layouts across devices.
