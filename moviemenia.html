MovieMenia

A modern full-stack movie discovery platform where users can explore movies, save favorites, receive AI-powered recommendations, and enjoy a smooth, responsive experience.

Live Demo GitHub

Project Overview

MovieMenia is a full-stack web application developed to provide users with an engaging platform for discovering movies. The application combines modern UI design, secure authentication, personalized watchlists, and AI-powered movie recommendations to create a seamless user experience.

Key Features

  • User Authentication (JWT)
  • Movie Search
  • Trending Movies
  • Movie Details
  • Favorites / Watchlist
  • AI Movie Recommendation
  • Responsive Design

Technology Stack

React Node.js Express MongoDB JWT Axios Tailwind CSS Gemini API

System Architecture

The frontend communicates with a REST API built using Express.js. User authentication is secured with JWT, while MongoDB stores user accounts, watchlists, and preferences. The recommendation engine analyzes user interests and generates personalized movie suggestions using AI.

Challenges

  • Managing authentication securely.
  • Integrating external movie APIs.
  • Generating personalized recommendations.
  • Maintaining responsive layouts across devices.

What I Learned

  • Designing scalable MERN architecture.
  • State management in React.
  • REST API development.
  • Authentication using JWT.
  • Prompt engineering for AI recommendations.
  • Deployment using Render.
